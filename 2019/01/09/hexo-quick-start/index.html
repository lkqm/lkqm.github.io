Hexo 搭建个人博客

路遥之玛丽
2019-01-09
tools

搭建原理

hexo是依赖于nodejs的博客框架(软件)，然后将文章(markdown格式)生成静态文件(html、css、js…)。再把这些生成的文件放到一个github项目上，利用github pages(github提供的一种服务)就可以访问了。

所以利用hexo搭建博客前请确保已经安装: nodejs、git

本地搭建并发布

安装

执行如上shell命令, 启动成功访问地址: http://localhost:4000, 默认的主题不太好看，所以需要自己再安装主题。

主题

hexo的主题有很多，对比了官网所有的主题，最终选择了cactus。

  1. 安装主题, 执行cd mario6.me && git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus
  2. 应用主题, 修改站点配置文件(mario6.me/_config.yml), 其中的配置项: theme: cactus
  3. 重写生成, 并预览新主题hexo clean && hexo g && hexo s
